Crear cliente Informacion de cuenta Consultar ultimos movimientos Cuentas de un cliente Realizar una transferencia Transferencias pendientes

Para poder solicitar el alta de un cliente se debe enviar el siguiente request a nuestra API:

Método:
POST
URL:
http://...urlbancoralito.../rest/cliente
Parámetro:
{
"du":[entero],
"saldo":[entero],
"usuario":"[string]",
"contrasenia":"[string]",
"email":"[string]"
}

La misma también se puede realizar por interfaz de usuario desde el cajero electrónico del banco.

Para consultar la información de una cuenta se debe enviar el siguiente request a nuestra API:

Método:
GET
URL:
http://...urlbancoralito.../rest/cuenta/{du}
Parámetro:
{
"du":[entero] }

Para consultar los últimos movimientos de una cuenta se debe enviar el siguiente request a nuestra API:

Método:
GET
URL:
http://...urlbancoralito.../{idcuenta}/{cantidadMov}
Parámetro:
{
"cantidadMov":[entero],
"idcuenta":[entero] }

Para conocer todas las cuentas que tiene un cliente se debe enviar el siguiente request a nuestra API:

Método:
GET
URL:
http://...urlbancoralito.../rest/cuenta/listaCuentas/{du}
Parámetro:
{
"du":[entero] }

Para poder realizar una transferencia se debe enviar el siguiente request a nuestra API:

Método:
POST
URL:
http://...urlbancoralito.../rest/movimiento
Parámetro:
{
"idTipoMovimiento":[entero],
"cuentaOrigen":[entero],
"cuentaDestino":"[entero]",
"importe":"[entero]",
"fecha":"[date]"
}

Para que el banco Central pueda consultar las transferencias pendientes de envío a otros bancos se debe enviar el siguiente request a nuestra API:

Método:
GET
URL:
http://...urlbancoralito.../rest/movimiento/transferenciaspendientes
